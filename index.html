Entendendo Comunicação Client x Server

História da Web

O bloco inicial trouxe a história da web. Aqui foi abordado desde os primeiros computadores e seus criadores, até o surgimento da internet, finalizando com os grandes nomes primordiais para a tecnologia atual.

O que são Clients

Apresenta-se aqui a definição de Clients no Linguajar Web e logo em seguida aborda sobre os navegadores, os chamados browsers.

Após isso o professor discorre acerca das aplicações web, conceituando como:

"Aplicações Web são soluções criadas que possuem a internet como meio de comunicação entre Client e Server não sendo necessária sua instalação."

O que são Servers

Nesta etapa há a definição de servidores, seus tipos e funcionalidades. Abordando por fim sobre a hospedagem de sites, necessária para a publicação dos dados na internet.

Linguagem de Programação

No último tópico são citadas as diversas linguagens de programação. As mais utilizadas para o server-side, assim como as típicas do client-size. Esclarecendo por fim a polêmica questão sobre HTML ser ou não uma linguagem.
